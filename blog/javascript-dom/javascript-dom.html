HTML, CSS and JavaScript

These three technologies work together to create beautiful and dynamic web applications. HTML stands for HyperText Markup Language and is responsible for all the content of the user interface, such as, buttons, paragraphs,images and text. CSS stands for Cascading Style Sheets and with this tool we can create beautiful webpages, because CSS gives you the power to style all the content. And last but not least, JavaScript is a programming language that allows you to add interactive effects and functionalities to the webpage.

To have a better comprehension of what all that means, let's think about the human body.

So in this analogy, HTML would be the structure of the body. CSS would be our appearance, like our clothes and hair color for example. Now JavaScript would be our nervous system and muscles. It gives everything movement and dynamic abilities.

Control flow and Loops

Control flow is the order in which the computer executes statements in a script. Code is run in order from the first line to the last line in the file, unless the computer runs across structures that change the control flow, such as conditionals and loops.

Loops are a fundamental part of JavaScript language, they are used to automate repetitive tasks based on a certain condition.

DOM

DOM stands for Document Object Model and it is a structured representation of HTML documents. It allows JavaScript to access HTML elements and styles in order to manipulate them in complex ways.

The DOM is not part of JavaScript, it is actually part of something called web API's. And API's are like libraries that the browser implement and we can then access using JavaScript.

Arrays vs Objects

Arrays and Objects are data structures, which is a collection of data values.

Arrays behave like a big container and it can hold as many values as you want. It can also accept values of any type.

Arrays are zero based, and this basically means that the first item of an array is at the position zero. So if you want to retrieve an element in the array you need to specify the index position and you can do this by using bracket notation.

Now, just like arrays, objects are used to essentially group different variables that really belong together. The big difference between them, is that in objects the order of these values does not matter at all when you want to access them.

There is two ways to retrieve an element in a object. You can use either bracket notation or dot notation.
